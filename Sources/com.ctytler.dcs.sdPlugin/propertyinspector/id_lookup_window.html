Type Descriptions:
BTN: Button - recommend using button with press/release set to the limit max/min
TUMB: Rotary/Switch - use increment to advance within range, or switch to select two values
LEV: Axis - use increment to advance within range, increment value can be adjusted for sensitivity. Some LEV items will need to use the momentary button with the "Send on Release" disabled.
